# Vegaa Icons

## Rocket Icon Files

Place your rocket icon files here:

- **`rocket-icon.png`** - Main logo/icon (recommended: 200x200px or larger)
- **`rocket-icon.svg`** - Scalable vector version (optional but recommended)
- **`favicon.ico`** - Favicon file (16x16, 32x32, 48x48 sizes)

## Usage

The icon will be used in:
- ✅ Navbar logo
- ✅ Browser favicon
- ✅ Landing page hero section
- ✅ Social media preview cards
- ✅ Anywhere Vegaa branding is needed

## File Formats

- **PNG**: Best for logos and general use
- **SVG**: Best for scalable graphics (recommended)
- **ICO**: Required for favicon (can be generated from PNG)

## Generating Favicon from PNG

If you have `rocket-icon.png`, you can generate `favicon.ico` using online tools:
- https://favicon.io/favicon-converter/
- https://www.favicon-generator.org/

Or use ImageMagick:
```bash
convert rocket-icon.png -resize 32x32 favicon.ico
```
